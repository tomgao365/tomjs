# tomjs

前端项目代码规范整理，方便快速搭建项目

## 使用说明

以 `pnpm` 为例

### 通用

```bash
pnpm add -D eslint prettier husky lint-staged @commitlint/cli @tomjs/eslint @tomjs/prettier @commitlint/cli @tomjs/commitlint @tomjs/tsconfig
```

### style

```bash
pnpm add -D stylelint @tomjs/stylelint
```
